腾讯Mysql数据库基础版无法上传2048kb大小sql文件问题

腾讯云
2018-03-15

产品类型：腾讯云Mysql数据库基础版

问题：

1.phpmyadmin后台上传数据库文件不支持大于2M的文件。

2.因为数据库是基础版本无法使用控制台的导入功能

3.因为数据库是基础版本无法使用数据迁移，数据迁移不支持基础版本

解决办法：

可以使用以下命令进行上传或者备份：

备份：mysqldump -h IP -P Port -u root -p dbname > XXX.sql
恢复：mysql -h IP -P Port -u root -p dbname < XXX.sql
