Sublime Text 3 安装,注册,设置中文教程

Sublime Text 3
2019-08-06

最近安装了sublime text3, 为了以后安装不再麻烦, 留个笔记. 同时希望能帮助有需要的你.

(过程有点简洁,照著一步步操作就行)

下载地址
下载地址 : http://www.sublimetext.com/ (官网)

安装 package control

  • 按 ctrl +~键,调出控制台

  • 在控制台裡输入:

import  urllib.request,os;pf='Package Control.sublime-package';ipp=sublime.installed_packages_path();urllib.request.install_opener(urllib.request.build_opener(urllib.request.ProxyHandler()));open(os.path.join(ipp,pf),'wb').write(urllib.request.urlopen('http://sublime.wbond.net/'+pf.replace(' ','%20')).read())

然后回车.看见在preferences裡有package control就说明安装成功了

设置中文

preferences->package control->搜寻框输入install->选择install package并回车->查询chinese->选择ChineseLocalization外挂

//安装后会自动把字元变为中文,可以在help裡的Language设定繁体,日语或者英语

安装过程亲测有效,软件版本为sublime text3
